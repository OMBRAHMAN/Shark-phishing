# 🦈 OM-SHARK v2.3.5

**OM-SHARK** ek powerful aur automated phishing tool hai jise Security Testing aur Educational purposes ke liye banaya gaya hai. Isme aapko 30 se zyada premium phishing templates milte hain.

## 🚀 Features
* ✅ **30+ Templates:** Facebook, Instagram, Google, FreeFire, PUBG, etc.
* ✅ **Easy to Use:** No complex commands, just select the number.
* ✅ **Lightweight:** Fast installation and low storage usage.
* ✅ **Auto-Save:** Victim data direct `results.txt` mein save hota hai.
